URL Field

An Angular directive or jQuery plugin that allows a user to enter a URL.

Angular

Dependencies Tooltip UI Bootstrap Storage Selector

The directive requires the use of the url attribute which configures the binding of the value for the URL.

Self-validation occurs on the input field value and if deemed invalid an error message displays. Additionally, a checkbox is provided to either opt in or out of validation.

Additionally, the Storage Selector component is available to pick a file from Storage, which fills the input field with the URL of the file selected. To do so, the directive requires the use of the company-id attribute which is a local scope property of the directive. The ID for the currently selected Company can be found in the URL where the ID is everything to the right of "company=", for example:

http://rva.risevision.com/#COMPANY_MANAGE/company=XXXXXXXX-XXXX-XXXX-XXXXXXXXXXXX

It should also be noted that the URL Field component automatically configures the appropriate URL to use for the HTML file selected from Storage so that it does not trigger a browser download.

You can optionally hide the label for URL Field by using the hideLabel attribute like so hideLabel="hideLabel". Similarly, the Storage Selector can be hidden using the hideStorage attribute like so hideStorage="hide".

The jQuery plugin version provides public methods getUrl(), setUrl(), and validateUrl() to use as you need.

Upon calling validateUrl(), if deemed invalid a checkbox is provided to either opt in or out of validation.

Note: An error message is not displayed in the jQuery version.
